HTML

HTML stands for Hyper Text Markup Language.

This is a tag based language where tags are created with greater than and less than signs

For Example

<p> this is a paragraph </p>

There are special charachters such as &lt; would be a less than sign. 


def fun(coding):
    print(coding, "is fun")

fun("HTML")

Semantic Elements

  • <header> is usually at the top of the page
  • <footer> is usually at the bottom of the page
  • See W3Schools for the rest

Inline vs Block

Inline means that it is inline with the text

Block level forces a new line

Block level: div, p, h1-5, li

Inline: span, img, a, strong, em
